# 🍿 간식 칼로리 기여 현황

아직 기여자가 없습니다. 첫 번째 간식을 추가해보세요!

## 🙋 참여 방법

1. 이 저장소를 **Fork** 하세요
2. `contributions/` 폴더에 `{본인_GitHub_ID}.txt` 파일을 만드세요
3. 파일에 간식명과 개수를 한 줄씩 입력하세요:
   ```
   오예스 12개
   포카칩 2봉지
   콜라 3캔
   새우깡 1봉
   ```
4. **Pull Request** 를 열면 자동으로 칼로리가 계산되어 랭킹에 반영됩니다 🎉

> 파일을 수정해서 PR을 보내면 기존 기여가 대체됩니다.
